Serviço de eco via UDP

Atualizado em 16/04/2019 

(i) Modifique o programa echoserver-udp.py de forma a torná-lo um servidor iterativo, 
faça login na xaveco.lab.ic.unicamp.br via ssh e execute-o  lá.

(ii) Modifique o programa echoclient-udp.py de forma a ecoar um arquivo texto de forma semelhante 
ao do programa echoclient-tcp.py, acrescentando um contador do número de linhas ecoadas pelo servidor,
assim como medida do tempo de execução (use o módulo time para isto).

(iii)  Execute num computador da IC305 o seu  echoclient-udp.py, passando na linha de comando o arquivo
 /etc/services. Faça 2 testes: um exibindo no vídeo cada linha ecoada e outro exibindo apenas o 
contador de linhas. Verifique se a contagem de linhas bate coma a do comando wc -l /etc/services

(iv) Opcional: faça o mesmo teste com o serviço de eco via TCP.
